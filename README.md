# Python Programming and Design Principles Assignments

This repository contains Python projects completed as part of the Programming and Design Principles 5N2927 module.

The projects helped me practise object-oriented programming, problem solving, input validation, testing, and documentation.

## Projects

### SD1 – IPv4 Address Validator and Classifier

A menu-driven Python program that validates IPv4 addresses and identifies their IP class.

Skills demonstrated:

* Object-Oriented Programming (OOP)
* Input validation
* String manipulation
* Conditional logic
* Menu-driven programs

### SD2 – Calculator Application

A Python calculator program created to practise program design, user input handling, and arithmetic operations.

Skills demonstrated:

* User input handling
* Arithmetic operations
* Conditional statements
* Program design
* Testing and debugging
