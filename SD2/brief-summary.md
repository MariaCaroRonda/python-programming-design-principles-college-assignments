# Brief Summary: SD2 Calculator Application

## Module

Programming and Design Principles 5N2927

## Assignment Title

Skills Demonstration 2: Design of a calculator application.

## Task Summary

The task was to design, code, and test a simple calculator application in Python.

The calculator needed to allow the user to:

- Add two numbers
- Subtract one number from another
- Multiply two numbers
- Divide one number by another
- Return to the main menu
- Exit the program

The assignment also included a team participation element, where each student needed to show evidence of their own role and contribution.

## Main Requirements

The assignment required:

- A working calculator application
- A user-friendly menu interface
- Clear prompts and instructions
- Addition, subtraction, multiplication, and division operations
- Validation for numeric input
- Handling for division by zero
- A flowchart
- Pseudocode
- A complete data dictionary
- Testing evidence with screenshots
- Clearly commented and indented source code
- Evidence of individual and group work
- Evidence of good team participation

## Implementation Summary

My solution uses a `Calculator` class with separate methods for each calculator operation.

The program includes methods to:

- Display the calculator menu
- Ask the user for two valid numbers
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers

The program uses a loop to keep showing the menu until the user chooses to exit. It also uses input validation to prevent invalid number entries and handles division by zero using an exception.
